# random-check-in

AI-generated React Native app built in the cloud.

## Features
- Random location generator
- Save favorite spots
- Share check-ins
- Check-in history log
- Map view

## How to Build
1. Go to Actions tab
2. Click "Run workflow"
3. Download APK from Artifacts
